Draw your own graph

The viewport is a live sketchpad. Build a graph directly, or load a preset and press Run Dijkstra.

  • Add a vertex double-click empty canvas
  • Set the source double-click an existing vertex
  • Add an edge shift-drag from one vertex to another
  • Move something drag it around
  • Delete something click it, then press the Delete key (not Backspace)
  • Numeric subscript put an underscore before the number (like "S_0")
  • Greek letter put a backslash before it (like "\beta")
  • Trace a path after running, click a vertex to see its shortest path
  • No mouse? use the Graph Editor panel to add or remove vertices and edges by name
