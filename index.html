Instructions

The big white box above is the where you can draw vertices and edges. Here's how to use it:

  • Add a vertex (node): double-click on the canvas
  • Add an edge: shift-drag on the canvas from one vertex to another
  • Move something: drag it around
  • Delete something: click it and press the delete key (not the backspace key)
  • Make source vertex: double-click on an existing vertex
  • Type numeric subscript: put an underscore before the number (like "S_0")
  • Type greek letter: put a backslash before it (like "\beta")

This was made in HTML5 and JavaScript using the canvas element.
